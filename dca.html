DCA Curriculum Details

Core Modules

Fundamentals of Computers

  • History and generations of computers.
  • Types of computers: Analog, Digital, Hybrid.
  • Computer components: CPU, input/output devices, and memory.

Operating Systems

  • Introduction to Windows and Linux.
  • File and folder management.
  • Control Panel settings and troubleshooting basics.

Microsoft Office Tools

  • MS Word: Document creation, editing, and formatting.
  • MS Excel: Formulas, functions, and data visualization using charts.
  • MS PowerPoint: Creating presentations with animations and transitions.
  • MS Access: Basics of databases and managing records.

Internet and Web Basics

  • Introduction to the internet and search engines.
  • Browsing, managing emails, and downloading/uploading files.
  • Basics of online collaboration tools (e.g., Google Drive).

Basics of Programming

  • Introduction to programming concepts.
  • Basics of languages like C, Python, or JavaScript (depending on the institute).

Multimedia Tools

  • Basics of graphic design using tools like Photoshop or Canva.
  • Introduction to video and audio editing software.

Advanced Modules

Database Management Systems (DBMS)

  • Introduction to databases and their importance.
  • Using SQL for creating, modifying, and querying databases.

Web Designing Basics

  • Introduction to HTML and CSS.
  • Designing simple static web pages.
  • Basics of responsive design.

Tally ERP and Accounting Software

  • Basics of Tally for managing accounts.
  • Introduction to GST calculations and financial reporting.

Cybersecurity Basics

  • Introduction to cybersecurity principles.
  • Online safety tips, secure passwords, and antivirus usage.
  • Awareness about phishing, scams, and malware.

Soft Skills and Practical Training

Soft Skills Development

  • Communication skills (verbal and written).
  • Time management and teamwork skills.

Project and Practical Work

  • Creating a document in MS Word or designing a small presentation.
  • Developing a simple HTML-based webpage.
  • Practical exercises on database creation using SQL.

Additional Opportunities

Workshops

  • Hands-on training in tools like Photoshop, Tally, or MS Excel.
  • Introduction to emerging trends like cloud storage or digital marketing.

Internships

  • Optional internships to gain industry exposure.
