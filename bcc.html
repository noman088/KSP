BCC Curriculum

Module 1: Introduction to Computers (2 Weeks)

Basics of Computers

  • History and generations of computers.
  • Components of a computer: Hardware and Software.
  • Input and output devices.

Fundamentals of Operating Systems

  • Introduction to operating systems (Windows/Linux).
  • Booting process and desktop environment.
  • File and folder management.

Basic Troubleshooting

  • Common hardware and software issues.
  • Using Control Panel and basic system settings.

Module 2: Office Automation (4 Weeks)

Microsoft Word

  • Creating, editing, and formatting documents.
  • Working with tables, images, and page setup.
  • Mail merge and document security.

Microsoft Excel

  • Basics of spreadsheets: rows, columns, and cells.
  • Applying formulas and functions.
  • Data analysis and creating charts.

Microsoft PowerPoint

  • Creating and editing presentations.
  • Slide design, layouts, animations, and transitions.
  • Delivering and printing presentations.

Module 3: Internet and Web Browsing (2 Weeks)

Internet Basics

  • Understanding the internet, browsers, and search engines.
  • Safe browsing practices and privacy awareness.
  • Downloading/uploading files and managing bookmarks.

Email Management

  • Creating and managing email accounts.
  • Composing, replying, and forwarding emails.
  • Handling attachments and spam management.

Introduction to Social Media

  • Overview of platforms like Facebook, Twitter, and LinkedIn.
  • Best practices for personal and professional use.

Module 4: Digital Financial Literacy (1 Week)

Digital Payments

  • Introduction to UPI, mobile wallets, and internet banking.
  • How to make secure transactions online.
  • Awareness about digital frauds and cybersecurity.

E-Governance Services

  • Accessing services like Aadhaar, PAN, and Passport online.
  • Using government portals for bill payments and registrations.

Module 5: Basics of Multimedia and Graphic Tools (2 Weeks)

Introduction to Graphics

  • Basics of graphic design using tools like MS Paint or Canva.
  • Simple image editing and creating posters or banners.

Audio and Video Tools

  • Using basic audio and video editing software.
  • Creating short presentations or clips.

Module 6: Cybersecurity Basics (1 Week)

Cyber Safety

  • Creating strong passwords and managing them.
  • Identifying phishing, malware, and scams.
  • Installing and using antivirus software.

Social Media Security

  • Understanding privacy settings.
  • Avoiding oversharing and managing secure connections.

Module 7: Practical Project and Assessment (1 Week)

Final Practical Project

  • Creating a professional resume in MS Word.
  • Designing a basic budget plan in MS Excel.
  • Preparing a short presentation using MS PowerPoint.

Assessment and Certification

  • Theory exam: Basic computer concepts.
  • Practical exam: Tasks using Office tools and the internet.

Learning Outcomes

  • Proficiency in basic computer operations and office applications.
  • Capability to browse the internet safely and manage emails.
  • Awareness of cybersecurity practices and digital financial tools.
  • Hands-on experience with productivity tools like MS Office.
